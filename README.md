# Swiggy Sales Data Analysis Dashboard

## 📊 Project Overview

This project is based on Swiggy sales data analysis. The dashboard is created in Microsoft Excel to analyze sales performance, customer ratings, orders, and sales trends across different cities and states in India.

The dashboard provides interactive insights using charts, KPIs, slicers, and maps.

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts
* Data Cleaning
* Data Visualization

## 📈 Dashboard KPIs

The dashboard shows the following key performance indicators:

* Total Sales: ₹53.01M
* Average Ratings: 4.34
* Average Order Value: ₹268.51
* Ratings Count: 5.59M
* Total Orders: 197.43K

## 📊 Dashboard Insights

The dashboard provides the following insights:

* Monthly Sales Trend
* Daily Sales Trend
* Weekly Sales Trend
* Sales by Food Type (Veg / Non-Veg)
* Sales by State (Map Visualization)
* Top 5 Cities by Sales
* Quarterly Sales, Ratings, and Orders
* Filter by Months, Restaurant Name, and Category

## 🎯 Project Objective

The objective of this project is to perform data analysis on Swiggy sales data and create an interactive dashboard to understand sales trends, customer behavior, and business performance.
